# Feuerwehr Verwaltung Pro

Echtzeit-Verwaltungssystem für Feuerwehren, basierend auf Google Sheets als Backend.

### Features
- **Live-Beförderungscheck:** Prüft automatisch Dienstzeit und Lehrgänge gegen die Laufbahnverordnung.
- **Einsatz-Abrechnung:** Berechnet Erstattungen und summiert Stunden automatisch.
- **Side-Panel:** Digitale Personalakte mit Click-to-Action (Anruf/Maps).
- **Stichtag-Simulation:** Prüfe Beförderungen für beliebige Daten in der Zukunft.

### Setup
1. Google Sheet als CSV freigeben.
2. URL in `js/service.js` eintragen.
3. Auf GitHub Pages oder beliebigem Webserver hosten.
